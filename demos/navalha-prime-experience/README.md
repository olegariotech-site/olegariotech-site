# Navalha Prime Barber Club — Demo Olegario Tech

Demo premium criada para apresentar a experiência que a Olegario Tech pode entregar para barbearias da região de Valinhos/Campinas.

> O AppBarber, Trinks ou WhatsApp cuida da agenda. A Olegario Tech cria a primeira impressão premium que faz o cliente querer agendar.

## Status

Estrutura inicial criada em HTML, CSS e JavaScript puro, usando GSAP e Lenis via CDN.

## Importante

Este é um projeto demonstrativo.

Nome, profissionais, avaliações, telefone e endereços são fictícios. Não usar nomes, logos, endereços ou dados reais de barbearias existentes sem autorização.
